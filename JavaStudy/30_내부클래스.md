# 내부 클래스
클래스 안에 클래스를 중첩하는 이유: 가독성, 유지보수성

예시:
```java
class OuterClass {
    int x = 10;

    class InnerClass {  // 내부클래스
        int y = 5;
    }
}

public class Main {
    public static void main(String[] args) {
        OuterClass outer = new OuterClass();
        OuterClass.InnerClass inner = outer.new InnerClass();
        System.out.println(inner.y + outer.x);  // 결과: 15 (5 + 10)
    }
}
```
## private
일반 클래스와 달리, 내부 클래스는 `private`나 `protected` 키워드를 사용할 수 있다.
```java
class Outerclass {
    int x = 10;

    private class InnerClass {
        int y = 5;
    }
}

publi class Main {
    public static void main(String[] args) {
        OuterClass outer = new OuterClass();
        OuterClass.InnerClass inner = outer.new InnerClass();   // 에러 발생, '.'으로 접근 불가
        System.out.println(inner.y + outer.x);
    }
}
```

## static
```java
class OuterClass {
    int x = 10;

    static class InnerClass {
        int y = 5;
    }
}

public class Main {
    public static void main(String[] args) {
        OuterClass.InnerClass myInner = new OuterClass.InnerClass();
        System.out.println(myInner.y);  // 결과: 5
    }
}
```
