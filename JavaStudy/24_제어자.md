# Modifier 제어자
## 접근제어자(Access modifier)
접근권한 제어
### 클래스
|제어자|설명|
|-|-|
|`public`|다른 클래스에서도 클래스에 접근할 수 있음|
|_디폴트_|같은 패키지 내의 클래스에만 접근 가능|
### 속성, 메서드, 생성자
|제어자|설명|
|-|-|
|`public`|다른 클래스에서 해당 코드 접근 가능|
|`private`|할당된 클래스에서만 접근 가능|
|_디폴트_|같은 패키지 내에서만 접근 가능|
|`protected`|같은 패키지나 서브클래스 내에서만 접근 가능|
```java
/* public, private 예시 */
class Person{
    public String name = "John";    //public
    private int age = 30;           //private
}

public class Main {
    public static void main(String[] args) {
        Person p = new Person();
        System.out.println(p.name); //제대로 동작
        System.out.println(p.age);  //에러 - 접근불가
    }
}
```

## 비접근제어자(일반제어자, Non-access modifier)
접근권한을 제어하지 않지만, 기타 다른 기능 제공
### `final`
선언한 값을 재정의하는 것을 원치 않는 경우
### `static`
`public`과 달리, 객체를 생성하지 않고도 접근 가능
```java
public class Main {
    static void myStaticMethod(){
        System.out.println("Static 메서드는 객체를 생성하지 않아도 호출 가능");
    }

    public static void main(String[] args){
        myStaticMethod();
        // static이 아니었으면, 
        // Main myObj = new Main();를 했어야 함
    }
}
```
### `abstract`
`abstract` 메서드는 반드시 추상클래스(abstract class)에 속해야 하며, body를 갖지 않는다. body는 서브클래스에서 제공된다.
> 추상 클래스: 객체를 직접 만들 수 없는 클래스
- 공통된 기능이나 구조를 정의하지만, 구체적인 구현은 하위 클래스(서브클래스)에서
- 클래스 선언 앞에 `abstract`
- 직접 `new` 키워드로 객체를 생성할 수 없음
- **상속을 통해서만 사용 가능!!!**
> 추상 메서드: 하위 클래스에서 반드시 재정의(override)해야 하는 메서드
- 메서드 선언에 `abstract`
- 몸체({})가 없음. 세미콜론으로 끝남
- 하위 클래스가 반드시 구현되어야 함
```java
/* Main.java */
//추상클래스 Main
abstract class Main {
    public String fname = "John";
    public int age = 24;
    public abstract void study();   // 추상메서드
}

//서브클래스 Student
class Student extends Main {
    public int graduationYear = 2025;
    public void study() {   //추상메서드의 body
        System.out.println("Study all day");
    }
}
```
```java
/* Second.java */
class Second {
    public static void main(String[] args) {
        Student myObj = new Student();

        System.out.println("Name: " + myObj.fname);
        System.out.println("Age: " + myObj.age);
        System.out.println("Graduation year: " + myObj.graduationYear);
        myObj.study();  //추상메서드 호출
    }
}
```
